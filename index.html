Basic CSS codes To Remember!

Color related content
Code Example Acronym meaning Explanation
Hexadecimal color: #fff The hash symbol tells the browser that the color is in a hexadecimal value! This is a very common way of setting colors to a browser, usualy taking three, six ou eight digits going from 0 to F (a, b, c, d, e and f are equal to 10, 11, 12, 14, 15 and 16 respectively)
RGB color: rgb(255, 255, 255) The rgb stands for red, green and blue. This is another way to set colors in a browser, using three values for red, green and blue. Each value goes from 0 to 255.
HSL color: hsl(0, 100%, 50%) The hsl stands for hue, saturation and lightness. This is a way to set colors in a browser, using three values for hue (0 to 360), saturation (0 to 100%) and lightness (0 to 100%).

Text related content
Code Example Acronym meaning Explanation
Font Family font-family: Arial, sans-serif; The font-family property sets the font family for a text. This property is used to specify a list of font family names and/or generic font family names. The browser then tries to find a font that matches the specified font family names.
Text Size font-size: 16px; The font-size property sets the size of a font. This property is used to specify the font size in pixels, ems, or rems.
Text Color color: #000; The color property sets the color of an element's text. This property is used to specify the color of the text, using hexadecimal, RGB, or HSL values, just like described above!
